## Overview

The Account API (`/v2/account`) manages your OFAuth account settings, webhook configurations, and provides access to connection data. This API is essential for receiving real-time notifications about connection events and managing your integration settings.

## Base URL

Base URL: `https://api.ofauth.com/v2/account`.

## Authentication

Include your OFAuth API key in the `apikey` header on every Account API request.

## Webhooks

Webhook configuration, event payloads, and delivery guidance are documented in the dedicated [Webhooks](/guides/webhooks) guide.

## Connection Management

### List Connections

Retrieve all active connections for your account.

<Expandable title="GET /connections">

- **Purpose**: List connections associated with your organization.
- **Endpoint**: `GET /v2/account/connections`
- **Query parameters**:

| Parameter | Type    | Description                                           |
| --------- | ------- | ----------------------------------------------------- |
| `limit`   | integer | Maximum number of connections to return (default: 50) |
| `offset`  | integer | Number of connections to skip (default: 0)            |
| `status`  | string  | Filter by status: `active`, `expired`, `revoked`      |

- **Response**: Returns a paginated list of connection objects along with pagination metadata (total, limit, offset, hasMore).

</Expandable>

### Get Connection Details

Retrieve detailed information about a specific connection.

<Expandable title="GET /connections/{connectionId}">

- **Purpose**: Retrieve full details for a specific connection.
- **Endpoint**: `GET /v2/account/connections/{connectionId}`
- **Response**: Returns a single connection object with metadata such as status, timestamps, client reference ID, and any profile information collected during linking.

</Expandable>

### Revoke Connection

Manually revoke a connection to immediately invalidate it.

<Expandable title="DELETE /connections/{connectionId}">

- **Purpose**: Revoke a connection immediately and prevent future Access API calls with it.
- **Endpoint**: `DELETE /v2/account/connections/{connectionId}`
- **Response**: Indicates the connection was revoked and provides updated status metadata.

</Expandable>
