## Official SDKs

OFAuth provides first-party SDKs generated from our OpenAPI spec, so types and methods always match the latest API. Pick the language that fits your stack.

<CodeGroup>
```bash TypeScript
npm install @ofauth/onlyfans-sdk
```

```bash Python
pip install onlyfans-sdk
```

```bash Go
go get github.com/ofauth-org/onlyfans-sdk-go
```

```bash C#
dotnet add package OnlyFans.SDK
```
</CodeGroup>

### Hello, OFAuth

Verify your API key with a single call:

<CodeGroup>
```typescript TypeScript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({ apiKey: 'your-api-key' });

const account = await client.account.whoami();
console.log(account.id, account.permissions);
```

```python Python
from onlyfans_sdk import OFAuthClient, account

client = OFAuthClient(api_key="your-api-key")

result = account.whoami(client)
print(result["id"], result["permissions"])
```

```go Go
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/ofauth-org/onlyfans-sdk-go"
)

func main() {
    client := ofauth.NewClient("your-api-key")

    acc, err := client.Whoami(context.Background())
    if err != nil {
        log.Fatal(err)
    }

    fmt.Println(acc.Id, acc.Permissions)
}
```

```csharp C#
using OFAuth;

var client = new OFAuthClient("your-api-key");

var account = await client.GetAsync<WhoamiResponse>("/v2/account/whoami");
Console.WriteLine($"{account.Id} {string.Join(", ", account.Permissions)}");
```
</CodeGroup>

## Feature Highlights

<CardGroup cols={3}>
  <Card title="Full Type Safety" icon="code">
    Every SDK is generated from the OpenAPI spec — response types, request params, and error codes are all typed.
  </Card>

  <Card title="Automatic Pagination" icon="arrows-rotate">
    Async iterators (TypeScript) and generators (Python) handle pagination automatically so you can `for-await` over any list.
  </Card>

  <Card title="Proxy Support" icon="globe">
    Call any OnlyFans API endpoint directly through the OFAuth proxy when the typed methods don't cover your use case.
  </Card>

  <Card title="Media Upload" icon="cloud-arrow-up">
    Upload images and videos with automatic chunking and progress callbacks built into every SDK.
  </Card>

  <Card title="Webhook Verification" icon="shield-check">
    Verify incoming webhook signatures to ensure events are authentic — included out of the box.
  </Card>

  <Card title="Error Handling" icon="triangle-exclamation">
    SDKs throw typed exceptions (`OFAuthAPIError`, `OFAuthError`, `APIError`, `OFAuthException`) with status, code, and details.
  </Card>
</CardGroup>

## SDK Comparison

| | TypeScript | Python | Go | C# |
| --- | --- | --- | --- | --- |
| **Package** | `@ofauth/onlyfans-sdk` | `onlyfans-sdk` | `onlyfans-sdk-go` | `OnlyFans.SDK` |
| **Repository** | [onlyfans-sdk-typescript](https://github.com/OFAuth-org/onlyfans-sdk-typescript) | [onlyfans-sdk-python](https://github.com/OFAuth-org/onlyfans-sdk-python) | [onlyfans-sdk-go](https://github.com/OFAuth-org/onlyfans-sdk-go) | [onlyfans-sdk-csharp](https://github.com/OFAuth-org/onlyfans-sdk-csharp) |
| **Init** | `createOFAuthClient()` | `OFAuthClient()` | `ofauth.NewClient()` | `new OFAuthClient()` |
| **Architecture** | Nested client with namespaces | Functional modules + client | 126+ typed methods, builder pattern | `GetAsync<T>(path)` with 473 model classes |
| **Types** | Inline types, full IntelliSense | Pydantic v2 models | Generated response structs | Generated classes with `JsonPropertyName` |
| **Pagination** | Async iterators (`for await`) | Generators (`for ... in`) | Manual (query params) | Manual (query params) |
| **Dependencies** | None (uses `fetch`) | `httpx` | Zero (standard library) | `System.Net.Http`, `System.Text.Json` |
| **Key Feature** | Nested namespaces (`client.access.listPosts()`) | Functional API (`posts.list_posts(client)`) | Context-based cancellation | DI support, `CancellationToken` |
