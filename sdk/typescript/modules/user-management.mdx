## Self Profile

### Get Creator Profile

Retrieve the authenticated creator's profile information.

```typescript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: 'your-api-key',
  connectionId: 'conn_xxx',
});

const profile = await client.access.self.get();
console.log(`${profile.name} (@${profile.username})`);
console.log(`Subscribers: ${profile.subscribersCount}`);
```

### Update Creator Profile

Update your profile information such as display name and bio.

```typescript
await client.access.self.update({
  body: {
    name: 'New Display Name',
    about: 'Updated bio information',
  },
});
```

**Permissions:** `profile:read` for get, `profile:write` for update.

## Users

### Get User by ID

Retrieve another user's profile by their ID.

```typescript
const user = await client.access.users.get({ userId: '987654321' });
console.log(`${user.name} (@${user.username})`);
```

**Permission required:** `profile:read`

### List Users by IDs

Look up multiple user profiles in a single request (up to 10 IDs).

```typescript
const result = await client.access.users.getList({
  userIds: '123,456,789',
});

for (const user of result) {
  console.log(user.id, user.username);
}
```

### Search Users

Search for performers/users by name or username.

```typescript
const results = await client.access.users.search({
  query: 'example',
  limit: 20,
});

for (const user of results.list) {
  console.log(user.username);
}
```

**Permission required:** `profile:read`

## Restrictions

### Restrict / Unrestrict a User

Restricted users can still see your profile but cannot interact with your content.

```typescript
// Restrict a user
await client.access.users.restrict({ userId: '987654321' });

// Unrestrict a user
await client.access.users.restrict2({ userId: '987654321' });
```

**Permission required:** `users:write`

### List Restricted Users

```typescript
const restricted = await client.access.users.getRestrict({
  limit: 20,
});

for (const user of restricted.list) {
  console.log(user.username);
}
```

### List Blocked Users

```typescript
const blocked = await client.access.users.getBlocked({
  limit: 20,
});

for (const user of blocked.list) {
  console.log(user.username);
}
```

## User Lists

User lists let you organize your fans into groups for targeted messaging and management.

### List User Lists

```typescript
const lists = await client.access.users.lists.list({
  limit: 20,
});

for (const list of lists.list) {
  console.log(list.id, list.name);
}
```

### Create, Update, and Delete User Lists

```typescript
// Create a list
const newList = await client.access.users.lists.create({
  body: { name: 'VIP Fans' },
});
console.log('Created list:', newList.id);

// Update a list
await client.access.users.lists.update({
  listId: '123',
  body: { name: 'Super VIP Fans' },
});

// Delete a list
await client.access.users.lists.delete({ listId: '123' });
```

**Permissions:** `lists:read` for listing, `lists:write` for create/update/delete.

### Add and Remove Users from Lists

```typescript
// Add a user to a list
await client.access.users.lists.createUsers({
  listId: '123',
  userId: '987654321',
});

// Remove a user from a list
await client.access.users.lists.deleteUsers({
  listId: '123',
  userId: '987654321',
});
```

### List Users in a List

```typescript
const listUsers = await client.access.users.lists.listUsers({
  listId: '123',
  limit: 20,
});

for (const user of listUsers.list) {
  console.log(user.username);
}
```

## Error Handling

All SDKs throw exceptions on API errors. Use try/catch to handle them:

```typescript
import { OFAuthAPIError } from '@ofauth/onlyfans-sdk';

try {
  const user = await client.access.users.get({ userId: '999' });
} catch (error) {
  if (error instanceof OFAuthAPIError) {
    console.error('API Error:', error.status, error.message);
    console.error('Error code:', error.code);
  } else {
    throw error;
  }
}
```
