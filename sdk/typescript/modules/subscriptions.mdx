## Subscribers

### List Subscribers

Get a paginated list of subscribers with optional filtering by status, date range, and search query.

```typescript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: 'your-api-key',
  connectionId: 'conn_xxx',
});

const subscribers = await client.access.subscribers.list({
  type: 'active',          // 'all' | 'active' | 'expired' | 'latest'
  limit: 50,
  offset: 0,
});

for (const sub of subscribers.list) {
  console.log(sub);
}
```

**Filter options for `type`:** `all` (default), `active`, `expired`, `latest`

When using `type=latest`, you can also pass `startDate`, `endDate`, and `latestType` (`total`, `new`, `renewals`):

```typescript
const latestSubs = await client.access.subscribers.list({
  type: 'latest',
  startDate: '2024-01-01',
  endDate: '2024-01-31',
  latestType: 'new',
  limit: 20,
});
```

### Paginating Subscribers

Each SDK provides idiomatic pagination. TypeScript uses async iterators, Python uses generators, and Go/C# use manual offset-based pagination.

```typescript
// Async iterator — handles pagination automatically
for await (const sub of client.access.subscribers.iterate({
  type: 'active',
  maxItems: 100,   // Stop after 100 items
  pageSize: 20,    // Items per page
})) {
  console.log(sub);
}
```

### Subscriber Actions

Update notes, apply discounts, and set custom names for individual subscribers.

```typescript
// Update subscriber note
await client.access.subscribers.setNote({
  userId: '12345',
  body: { text: 'VIP subscriber' },
});

// Apply discount to subscriber
await client.access.subscribers.setDiscount({
  userId: '12345',
  body: { discount: 50, months: 3 },
});

// Set custom display name
await client.access.subscribers.setCustomName({
  userId: '12345',
  body: { name: 'My Custom Name' },
});
```

## Subscriptions

Subscriptions represent people *you* are subscribed to or following (the inverse of subscribers).

### List Subscriptions

```typescript
const subscriptions = await client.access.subscriptions.list({
  type: 'active',          // 'all' | 'active' | 'expired'
  limit: 20,
  offset: 0,
});

for (const sub of subscriptions.list) {
  console.log(sub);
}

// Auto-paginate through all subscriptions
for await (const sub of client.access.subscriptions.iterate({
  type: 'active',
})) {
  console.log(sub);
}
```

### Subscription Counts

Get aggregated counts of subscriptions and subscribers by status.

```typescript
const counts = await client.access.subscriptions.getCount();
console.log('Subscriptions:', counts.subscriptions);
console.log('Subscribers:', counts.subscribers);
```

### Get Subscription by ID

```typescript
const subscription = await client.access.subscriptions.get({
  subscriptionId: 'sub_123',
});
```

### Subscription History

View the renewal and change history for a specific subscription.

```typescript
const history = await client.access.subscriptions.getHistory({
  subscriptionId: 'sub_123',
  all: '1',                // '0' | '1' | 'true' | 'false' — include all history
});
```

## Pagination Patterns

Each SDK handles pagination differently to match language idioms:

| Language | Pattern | Example |
|---|---|---|
| **TypeScript** | Async iterators (`for await...of`) | `client.access.subscribers.iterate({ maxItems: 100 })` |
| **Python** | Generators (`for...in`) | `subscribers.iter_subscribers(client, max_items=100)` |
| **Go** | Manual offset loop | Check `response.HasMore`, increment `offset` |
| **C#** | Manual offset loop | Check `response.HasMore`, increment `offset` |

<Tip>
  TypeScript and Python iterators handle pagination automatically — you just iterate and the SDK fetches pages behind the scenes. For Go and C#, use the `HasMore` flag and `offset` query parameter to manually page through results.
</Tip>
