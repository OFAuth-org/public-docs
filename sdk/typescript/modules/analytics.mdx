## Earnings Chart

Get time-series earnings data for a connection. Use `startDate` and `endDate` to specify a date range, and `by` to break down by earning type.

```typescript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: 'your-api-key',
  connectionId: 'conn_xxx',
});

const chart = await client.access.earnings.getChart({
  startDate: '2024-01-01',
  endDate: '2024-01-31',
  by: 'total',         // 'total' | 'messages' | 'tips' | 'stream' | 'post' | 'subscribes' | ...
  withTotal: true,
});
```

**`by` parameter options:** `total`, `messages`, `tips`, `stream`, `post`, `subscribes`, `tips_profile`, `tips_post`, `tips_chat`, `tips_stream`, `tips_story`, `ref`

## Transactions

List individual earnings transactions, optionally filtered by type.

```typescript
// List transactions
const transactions = await client.access.earnings.listTransactions({
  startDate: '2024-01-01',
  type: 'tips',            // 'subscribes' | 'chat_messages' | 'post' | 'stream' | 'tips'
  tipsSource: 'chat',      // 'chat' | 'post_all' | 'profile' | 'story' | 'stream'
});

// Auto-paginate through all transactions
for await (const tx of client.access.earnings.iterateTransactions({
  type: 'tips',
})) {
  console.log(tx);
}
```

## Chargebacks

List chargeback records with optional date range filtering.

```typescript
const chargebacks = await client.access.earnings.listChargebacks({
  startDate: '2024-01-01',
  endDate: '2024-01-31',
  limit: 20,
  offset: 0,
});

// Auto-paginate through all chargebacks
for await (const cb of client.access.earnings.iterateChargebacks({
  startDate: '2024-01-01',
  endDate: '2024-01-31',
})) {
  console.log(cb);
}
```

## Analytics

Content performance analytics are available across multiple categories. Each category offers a **chart** endpoint (time-series data) and a **top** endpoint (ranked content).

<Info>
  All analytics endpoints require the `analytics:read` permission.
</Info>

### Posts Analytics

```typescript
// Posts chart â€” time-series performance data
const postsChart = await client.access.analytics.posts.getChart({
  startDate: '2024-01-01',
  endDate: '2024-01-31',
  by: 'views',       // 'purchases' | 'posts' | 'tips' | 'views' | 'likes' | 'comments'
  withTotal: true,
});

// Top performing posts
const topPosts = await client.access.analytics.posts.getTop({
  by: 'purchases',   // 'purchases' | 'tips' | 'views' | 'likes' | 'comments'
  startDate: '2024-01-01',
  endDate: '2024-01-31',
  limit: 10,
});

// Stats for a specific post
const postStats = await client.access.analytics.posts.get({
  postId: '123456',
});
```

### Mass Messages Analytics

```typescript
// Mass messages chart
const mmChart = await client.access.analytics.massMessages.getChart({
  startDate: '2024-01-01',
  endDate: '2024-01-31',
  withTotal: true,
});

// List purchased mass messages
const purchased = await client.access.analytics.massMessages.getPurchased({
  limit: 20,
  offset: 0,
});

// List buyers for a specific mass message
const buyers = await client.access.analytics.massMessages.listBuyers({
  massMessageId: '456',
  limit: 20,
});

// Auto-paginate through all buyers
for await (const buyer of client.access.analytics.massMessages.iterateBuyers({
  massMessageId: '456',
})) {
  console.log(buyer);
}
```

const campaignsChart = await client.access.analytics.campaigns.getChart({

</Warning>
