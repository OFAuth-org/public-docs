<Info>
  **Prerequisites:** You need an [OFAuth account](https://app.ofauth.com) and an API key. Grab your key from [app.ofauth.com/platform/developers/keys](https://app.ofauth.com/platform/developers/keys).
</Info>

<Steps>

<Step title="Install the SDK">

Pick your package manager and install:

<CodeGroup>
```bash npm
npm install @ofauth/onlyfans-sdk
```

```bash pnpm
pnpm add @ofauth/onlyfans-sdk
```

```bash yarn
yarn add @ofauth/onlyfans-sdk
```
</CodeGroup>

</Step>

<Step title="Initialize the client">

Create a client with your API key:

```typescript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: process.env.OFAUTH_API_KEY!,
});
```

<Tip>
  Never hard-code your API key. Use environment variables (`OFAUTH_API_KEY`) or a secrets manager in production.
</Tip>

</Step>

<Step title="Verify your API key">

Call `whoami` to confirm your key works:

```typescript
import { createOFAuthClient, OFAuthAPIError } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: process.env.OFAUTH_API_KEY!,
});

try {
  const account = await client.account.whoami();
  console.log('Account ID:', account.id);
  console.log('Permissions:', account.permissions);
} catch (error) {
  if (error instanceof OFAuthAPIError) {
    console.error('API Error:', error.status, error.message);
  }
  throw error;
}
```

<Check>
  You should see your organization ID and permissions list. If you get a `401` error, double-check your API key.
</Check>

</Step>

<Step title="Fetch OnlyFans data">

Once you have a connection ID (from [Link](/quickstart#step-2-connect-an-onlyfans-account)), use the Access API to pull data from a connected OnlyFans account:

```typescript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: process.env.OFAUTH_API_KEY!,
  connectionId: 'conn_YOUR_CONNECTION_ID',
});

// Get the connected creator's profile
const profile = await client.access.self.get();
console.log('Username:', profile.name);

// Or pass connectionId per-request
const profile2 = await client.access.self.get({
  connectionId: 'conn_OTHER_CONNECTION',
});
```

<Check>
  You're now pulling live data from a connected OnlyFans account through OFAuth.
</Check>

</Step>

</Steps>
