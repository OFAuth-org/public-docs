## Self Profile

### Get Creator Profile

Retrieve the authenticated creator's profile information.

<CodeGroup>
```typescript TypeScript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: 'your-api-key',
  connectionId: 'conn_xxx',
});

const profile = await client.access.self.get();
console.log(`${profile.name} (@${profile.username})`);
console.log(`Subscribers: ${profile.subscribersCount}`);
```

```python Python
from onlyfans_sdk import OFAuthClient
from onlyfans_sdk import self as self_module  # 'self' is a Python keyword

client = OFAuthClient(api_key="your-api-key", connection_id="conn_xxx")

profile = self_module.list_selfs(client)
print(f"{profile['name']} (@{profile['username']})")
print(f"Subscribers: {profile['subscribersCount']}")
```

```go Go
ctx := context.Background()
client := ofauth.NewClient("your-api-key").
    WithConnectionID("conn_xxx")

profile, err := client.GetAccessSelf(ctx)
if err != nil {
    log.Fatal(err)
}

fmt.Printf("%s (@%s)\n", (*profile)["name"], (*profile)["username"])
```

```csharp C#
using OFAuth;

var client = new OFAuthClient("your-api-key")
    .WithConnectionId("conn_xxx");

var profile = await client.GetAsync<SelfResponse>("/v2/access/self");
Console.WriteLine($"{profile.Name} (@{profile.Username})");
Console.WriteLine($"Subscribers: {profile.SubscribersCount}");
```
</CodeGroup>

### Update Creator Profile

Update your profile information such as display name and bio.

<CodeGroup>
```typescript TypeScript
await client.access.self.update({
  body: {
    name: 'New Display Name',
    about: 'Updated bio information',
  },
});
```

```python Python
self_module.update_self(
    client,
    body={
        "name": "New Display Name",
        "about": "Updated bio information",
    },
)
```

```go Go
_, err := client.UpdateAccessSelf(ctx, struct {
    Name  string `json:"name"`
    About string `json:"about"`
}{
    Name:  "New Display Name",
    About: "Updated bio information",
})
if err != nil {
    log.Fatal(err)
}
```

```csharp C#
await client.PatchAsync<SelfResponse>(
    "/v2/access/self",
    body: new
    {
        name = "New Display Name",
        about = "Updated bio information"
    }
);
```
</CodeGroup>

**Permissions:** `profile:read` for get, `profile:write` for update.

## Users

### Get User by ID

Retrieve another user's profile by their ID.

<CodeGroup>
```typescript TypeScript
const user = await client.access.users.get({ userId: '987654321' });
console.log(`${user.name} (@${user.username})`);
```

```python Python
from onlyfans_sdk import users

user = users.get_users(client, user_id="987654321")
print(f"{user['name']} (@{user['username']})")
```

```go Go
user, err := client.GetAccessUsers(ctx, "987654321")
if err != nil {
    log.Fatal(err)
}

fmt.Printf("%s\n", (*user)["username"])
```

```csharp C#
var user = await client.GetAsync<UserResponse>("/v2/access/users/987654321");
Console.WriteLine($"{user.Name} (@{user.Username})");
```
</CodeGroup>

**Permission required:** `profile:read`

### List Users by IDs

Look up multiple user profiles in a single request (up to 10 IDs).

<CodeGroup>
```typescript TypeScript
const result = await client.access.users.getList({
  userIds: '123,456,789',
});

for (const user of result) {
  console.log(user.id, user.username);
}
```

```python Python
result = users.list_lists(client, user_ids="123,456,789")

for user in result:
    print(user["id"], user["username"])
```

```go Go
result, err := client.GetAccessUsersList(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "userIds": []string{"123,456,789"},
    },
})
if err != nil {
    log.Fatal(err)
}
```

```csharp C#
var result = await client.GetAsync<List<UserResponse>>(
    "/v2/access/users/list",
    query: new Dictionary<string, string?> { ["userIds"] = "123,456,789" }
);

foreach (var user in result)
{
    Console.WriteLine($"{user.Id}: {user.Username}");
}
```
</CodeGroup>

### Search Users

Search for performers/users by name or username.

<CodeGroup>
```typescript TypeScript
const results = await client.access.users.search({
  query: 'example',
  limit: 20,
});

for (const user of results.list) {
  console.log(user.username);
}
```

```python Python
results = users.list_searchs(client, query="example", limit=20)

for user in results["list"]:
    print(user["username"])
```

```go Go
results, err := client.AccessUsersSearch(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "query": []string{"example"},
        "limit": []string{"20"},
    },
})
if err != nil {
    log.Fatal(err)
}
```

```csharp C#
var results = await client.GetAsync<UsersSearchResponse>(
    "/v2/access/users/search",
    query: new Dictionary<string, string?>
    {
        ["query"] = "example",
        ["limit"] = "20"
    }
);

foreach (var user in results.List)
{
    Console.WriteLine(user.Username);
}
```
</CodeGroup>

**Permission required:** `profile:read`

## Restrictions

### Restrict / Unrestrict a User

Restricted users can still see your profile but cannot interact with your content.

<CodeGroup>
```typescript TypeScript
// Restrict a user
await client.access.users.restrict({ userId: '987654321' });

// Unrestrict a user
await client.access.users.restrict2({ userId: '987654321' });
```

```python Python
# Restrict a user
users.create_restrict(client, user_id="987654321")

# Unrestrict a user
users.delete_restrict(client, user_id="987654321")
```

```go Go
// Restrict a user
_, err := client.CreateAccessUsersRestrict(ctx, "987654321")

// Unrestrict a user
_, err = client.DeleteAccessUsersRestrict(ctx, "987654321")
```

```csharp C#
// Restrict a user
await client.PostAsync<object>("/v2/access/users/987654321/restrict");

// Unrestrict a user
await client.DeleteAsync<object>("/v2/access/users/987654321/restrict");
```
</CodeGroup>

**Permission required:** `users:write`

## User Lists

User lists let you organize your fans into groups for targeted messaging and management.

### List User Lists

<CodeGroup>
```typescript TypeScript
const lists = await client.access.users.lists.list({
  limit: 20,
});

for (const list of lists.list) {
  console.log(list.id, list.name);
}
```

```python Python
from onlyfans_sdk import user_lists

lists = user_lists.list_users_users_lists(client, limit=20)

for lst in lists["list"]:
    print(lst["id"], lst["name"])
```

```go Go
lists, err := client.ListAccessUsersLists(ctx, &ofauth.RequestOptions{
    Query: url.Values{"limit": []string{"20"}},
})
if err != nil {
    log.Fatal(err)
}

for _, list := range lists.List {
    fmt.Println(list.Id, list.Name)
}
```

```csharp C#
var lists = await client.GetAsync<UsersListsResponse>(
    "/v2/access/users/lists",
    query: new Dictionary<string, string?> { ["limit"] = "20" }
);

foreach (var list in lists.List)
{
    Console.WriteLine($"{list.Id}: {list.Name}");
}
```
</CodeGroup>

### Create, Update, and Delete User Lists

<CodeGroup>
```typescript TypeScript
// Create a list
const newList = await client.access.users.lists.create({
  body: { name: 'VIP Fans' },
});
console.log('Created list:', newList.id);

// Update a list
await client.access.users.lists.update({
  listId: '123',
  body: { name: 'Super VIP Fans' },
});

// Delete a list
await client.access.users.lists.delete({ listId: '123' });
```

```python Python
# Create a list
new_list = user_lists.create_users_users_lists(
    client,
    body={"name": "VIP Fans"},
)
print("Created list:", new_list["id"])

# Update a list
user_lists.update_users_users_lists(
    client,
    list_id="123",
    body={"name": "Super VIP Fans"},
)

# Delete a list
user_lists.delete_users_users_lists(client, list_id="123")
```

```go Go
// Create a list
newList, err := client.CreateAccessUsersLists(ctx, struct {
    Name string `json:"name"`
}{Name: "VIP Fans"})

// Update a list
_, err = client.UpdateAccessUsersLists(ctx, "123", struct {
    Name string `json:"name"`
}{Name: "Super VIP Fans"})

// Delete a list
_, err = client.DeleteAccessUsersLists(ctx, "123")
```

```csharp C#
// Create a list
var newList = await client.PostAsync<UserListResponse>(
    "/v2/access/users/lists",
    body: new { name = "VIP Fans" }
);
Console.WriteLine($"Created list: {newList.Id}");

// Update a list
await client.PatchAsync<UserListResponse>(
    "/v2/access/users/lists/123",
    body: new { name = "Super VIP Fans" }
);

// Delete a list
await client.DeleteAsync<object>("/v2/access/users/lists/123");
```
</CodeGroup>

**Permissions:** `lists:read` for listing, `lists:write` for create/update/delete.

## Error Handling

All SDKs throw exceptions on API errors. Use try/catch to handle them:

<CodeGroup>
```typescript TypeScript
import { OFAuthAPIError } from '@ofauth/onlyfans-sdk';

try {
  const user = await client.access.users.get({ userId: '999' });
} catch (error) {
  if (error instanceof OFAuthAPIError) {
    console.error('API Error:', error.status, error.message);
    console.error('Error code:', error.code);
  } else {
    throw error;
  }
}
```

```python Python
from onlyfans_sdk import OFAuthError

try:
    user = users.get_users(client, user_id="999")
except OFAuthError as e:
    print(f"API Error {e.status}: {e}")
    print(f"Error code: {e.code}")
```

```go Go
user, err := client.GetAccessUsers(ctx, "999")
if err != nil {
    if apiErr, ok := err.(*ofauth.APIError); ok {
        fmt.Printf("API Error %d: %s\n", apiErr.Status, apiErr.Message)
    } else {
        log.Fatal(err)
    }
}
```

```csharp C#
try
{
    var user = await client.GetAsync<UserResponse>("/v2/access/users/999");
}
catch (OFAuthException ex)
{
    Console.WriteLine($"API Error {ex.Status}: {ex.Message}");
    Console.WriteLine($"Error code: {ex.Code}");
}
```
</CodeGroup>
