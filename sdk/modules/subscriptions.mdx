## Subscribers

### List Subscribers

Get a paginated list of subscribers with optional filtering by status, date range, and search query.

<CodeGroup>
```typescript TypeScript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: 'your-api-key',
  connectionId: 'conn_xxx',
});

const subscribers = await client.access.subscribers.list({
  type: 'active',          // 'all' | 'active' | 'expired' | 'latest'
  limit: 50,
  offset: 0,
});

for (const sub of subscribers.list) {
  console.log(sub);
}
```

```python Python
from onlyfans_sdk import OFAuthClient, subscribers

client = OFAuthClient(api_key="your-api-key", connection_id="conn_xxx")

subs = subscribers.list_subscribers(client, type="active", limit=50)
for sub in subs["list"]:
    print(sub)
```

```go Go
subs, err := client.ListAccessSubscribers(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "type":  []string{"active"},
        "limit": []string{"50"},
    },
})
if err != nil {
    log.Fatal(err)
}
for _, sub := range subs.List {
    fmt.Printf("%v\n", sub)
}
```

```csharp C#
var subs = await client.GetAsync<SubscribersListResponse>(
    "/v2/access/subscribers",
    query: new Dictionary<string, string?>
    {
        ["type"] = "active",
        ["limit"] = "50"
    }
);
foreach (var sub in subs.List)
{
    Console.WriteLine(sub);
}
```
</CodeGroup>

**Filter options for `type`:** `all` (default), `active`, `expired`, `latest`

When using `type=latest`, you can also pass `startDate`, `endDate`, and `latestType` (`total`, `new`, `renewals`):

<CodeGroup>
```typescript TypeScript
const latestSubs = await client.access.subscribers.list({
  type: 'latest',
  startDate: '2024-01-01',
  endDate: '2024-01-31',
  latestType: 'new',
  limit: 20,
});
```

```python Python
latest_subs = subscribers.list_subscribers(
    client,
    type="latest",
    start_date="2024-01-01",
    end_date="2024-01-31",
    latest_type="new",
    limit=20,
)
```

```go Go
latestSubs, err := client.ListAccessSubscribers(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "type":       []string{"latest"},
        "startDate":  []string{"2024-01-01"},
        "endDate":    []string{"2024-01-31"},
        "latestType": []string{"new"},
        "limit":      []string{"20"},
    },
})
```

```csharp C#
var latestSubs = await client.GetAsync<SubscribersListResponse>(
    "/v2/access/subscribers",
    query: new Dictionary<string, string?>
    {
        ["type"] = "latest",
        ["startDate"] = "2024-01-01",
        ["endDate"] = "2024-01-31",
        ["latestType"] = "new",
        ["limit"] = "20"
    }
);
```
</CodeGroup>

### Paginating Subscribers

Each SDK provides idiomatic pagination. TypeScript uses async iterators, Python uses generators, and Go/C# use manual offset-based pagination.

<CodeGroup>
```typescript TypeScript
// Async iterator — handles pagination automatically
for await (const sub of client.access.subscribers.iterate({
  type: 'active',
  maxItems: 100,   // Stop after 100 items
  pageSize: 20,    // Items per page
})) {
  console.log(sub);
}
```

```python Python
# Generator — handles pagination automatically
for sub in subscribers.iter_subscribers(client, type="active", max_items=100, page_size=20):
    print(sub)
```

```go Go
// Manual pagination with offset
offset := 0
limit := 20

for {
    subs, err := client.ListAccessSubscribers(ctx, &ofauth.RequestOptions{
        Query: url.Values{
            "type":   []string{"active"},
            "limit":  []string{fmt.Sprintf("%d", limit)},
            "offset": []string{fmt.Sprintf("%d", offset)},
        },
    })
    if err != nil {
        log.Fatal(err)
    }

    for _, sub := range subs.List {
        fmt.Printf("%v\n", sub)
    }

    if !subs.HasMore {
        break
    }
    offset += len(subs.List)
}
```

```csharp C#
// Manual pagination with offset
int offset = 0;
int limit = 20;

while (true)
{
    var subs = await client.GetAsync<SubscribersListResponse>(
        "/v2/access/subscribers",
        query: new Dictionary<string, string?>
        {
            ["type"] = "active",
            ["limit"] = limit.ToString(),
            ["offset"] = offset.ToString()
        }
    );

    foreach (var sub in subs.List)
    {
        Console.WriteLine(sub);
    }

    if (!subs.HasMore) break;
    offset += subs.List.Count;
}
```
</CodeGroup>

### Subscriber Actions

Update notes, apply discounts, and set custom names for individual subscribers.

<CodeGroup>
```typescript TypeScript
// Update subscriber note
await client.access.subscribers.setNote({
  userId: '12345',
  body: { text: 'VIP subscriber' },
});

// Apply discount to subscriber
await client.access.subscribers.setDiscount({
  userId: '12345',
  body: { discount: 50, months: 3 },
});

// Set custom display name
await client.access.subscribers.setCustomName({
  userId: '12345',
  body: { name: 'My Custom Name' },
});
```

```python Python
# Update subscriber note
subscribers.set_note(client, user_id="12345", body={"text": "VIP subscriber"})

# Apply discount to subscriber
subscribers.set_discount(client, user_id="12345", body={"discount": 50, "months": 3})

# Set custom display name
subscribers.set_custom_name(client, user_id="12345", body={"name": "My Custom Name"})
```

```go Go
// Update subscriber note
_, err := client.SetAccessSubscribersNote(ctx, "12345", struct {
    Text string `json:"text"`
}{Text: "VIP subscriber"})

// Apply discount to subscriber
_, err = client.SetAccessSubscribersDiscount(ctx, "12345", struct {
    Discount int `json:"discount"`
    Months   int `json:"months"`
}{Discount: 50, Months: 3})
```

```csharp C#
// Update subscriber note
await client.PutAsync<object>(
    "/v2/access/subscribers/12345/note",
    body: new { text = "VIP subscriber" }
);

// Apply discount to subscriber
await client.PutAsync<object>(
    "/v2/access/subscribers/12345/discount",
    body: new { discount = 50, months = 3 }
);

// Set custom display name
await client.PutAsync<object>(
    "/v2/access/subscribers/12345/custom-name",
    body: new { name = "My Custom Name" }
);
```
</CodeGroup>

## Subscriptions

Subscriptions represent people *you* are subscribed to or following (the inverse of subscribers).

### List Subscriptions

<CodeGroup>
```typescript TypeScript
const subscriptions = await client.access.subscriptions.list({
  type: 'active',          // 'all' | 'active' | 'expired'
  limit: 20,
  offset: 0,
});

for (const sub of subscriptions.list) {
  console.log(sub);
}

// Auto-paginate through all subscriptions
for await (const sub of client.access.subscriptions.iterate({
  type: 'active',
})) {
  console.log(sub);
}
```

```python Python
from onlyfans_sdk import subscriptions

subs = subscriptions.list_subscriptions(client, type="active", limit=20)
for sub in subs["list"]:
    print(sub)

# Auto-paginate through all subscriptions
for sub in subscriptions.iter_subscriptions(client, type="active"):
    print(sub)
```

```go Go
subs, err := client.ListAccessSubscriptions(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "type":  []string{"active"},
        "limit": []string{"20"},
    },
})
for _, sub := range subs.List {
    fmt.Printf("%v\n", sub)
}
```

```csharp C#
var subs = await client.GetAsync<SubscriptionsListResponse>(
    "/v2/access/subscriptions",
    query: new Dictionary<string, string?>
    {
        ["type"] = "active",
        ["limit"] = "20"
    }
);
foreach (var sub in subs.List)
{
    Console.WriteLine(sub);
}
```
</CodeGroup>

### Subscription Counts

Get aggregated counts of subscriptions and subscribers by status.

<CodeGroup>
```typescript TypeScript
const counts = await client.access.subscriptions.getCount();
console.log('Subscriptions:', counts.subscriptions);
console.log('Subscribers:', counts.subscribers);
```

```python Python
counts = subscriptions.list_counts(client)
print("Subscriptions:", counts["subscriptions"])
print("Subscribers:", counts["subscribers"])
```

```go Go
counts, err := client.GetAccessSubscriptionsCount(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Subscriptions: %+v\n", counts.Subscriptions)
fmt.Printf("Subscribers: %+v\n", counts.Subscribers)
```

```csharp C#
var counts = await client.GetAsync<SubscriptionsCountResponse>(
    "/v2/access/subscriptions/count"
);
Console.WriteLine($"Subscriptions: {counts.Subscriptions}");
Console.WriteLine($"Subscribers: {counts.Subscribers}");
```
</CodeGroup>

### Subscription History

View the renewal and change history for a specific subscription.

<CodeGroup>
```typescript TypeScript
const history = await client.access.subscriptions.getHistory({
  subscriptionId: 'sub_123',
  all: '1',                // '0' | '1' | 'true' | 'false' — include all history
});
```

```python Python
history = subscriptions.list_historys(
    client,
    subscription_id="sub_123",
    all="1",
)
```

```go Go
history, err := client.GetAccessSubscriptionsHistory(ctx, "sub_123", &ofauth.RequestOptions{
    Query: url.Values{
        "all": []string{"1"},
    },
})
```

```csharp C#
var history = await client.GetAsync<SubscriptionHistoryResponse>(
    "/v2/access/subscriptions/sub_123/history",
    query: new Dictionary<string, string?> { ["all"] = "1" }
);
```
</CodeGroup>

## Pagination Patterns

Each SDK handles pagination differently to match language idioms:

| Language | Pattern | Example |
|---|---|---|
| **TypeScript** | Async iterators (`for await...of`) | `client.access.subscribers.iterate({ maxItems: 100 })` |
| **Python** | Generators (`for...in`) | `subscribers.iter_subscribers(client, max_items=100)` |
| **Go** | Manual offset loop | Check `response.HasMore`, increment `offset` |
| **C#** | Manual offset loop | Check `response.HasMore`, increment `offset` |

<Tip>
  TypeScript and Python iterators handle pagination automatically — you just iterate and the SDK fetches pages behind the scenes. For Go and C#, use the `HasMore` flag and `offset` query parameter to manually page through results.
</Tip>
