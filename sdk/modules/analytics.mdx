## Earnings Chart

Get time-series earnings data for a connection. Use `startDate` and `endDate` to specify a date range, and `by` to break down by earning type.

<CodeGroup>
```typescript TypeScript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: 'your-api-key',
  connectionId: 'conn_xxx',
});

const chart = await client.access.earnings.getChart({
  startDate: '2024-01-01',
  endDate: '2024-01-31',
  by: 'total',         // 'total' | 'messages' | 'tips' | 'stream' | 'post' | 'subscribes' | ...
  withTotal: true,
});
```

```python Python
from onlyfans_sdk import OFAuthClient, earnings

client = OFAuthClient(api_key="your-api-key", connection_id="conn_xxx")

chart = earnings.list_charts(
    client,
    start_date="2024-01-01",
    end_date="2024-01-31",
    by="total",
    with_total=True,
)
```

```go Go
earnings, err := client.GetAccessEarningsChart(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "startDate": []string{"2024-01-01"},
        "endDate":   []string{"2024-01-31"},
        "by":        []string{"total"},
        "withTotal": []string{"true"},
    },
})
if err != nil {
    log.Fatal(err)
}
```

```csharp C#
var chart = await client.GetAsync<EarningsChartResponse>(
    "/v2/access/earnings/chart",
    query: new Dictionary<string, string?>
    {
        ["startDate"] = "2024-01-01",
        ["endDate"] = "2024-01-31",
        ["by"] = "total",
        ["withTotal"] = "true"
    }
);
```
</CodeGroup>

**`by` parameter options:** `total`, `messages`, `tips`, `stream`, `post`, `subscribes`, `tips_profile`, `tips_post`, `tips_chat`, `tips_stream`, `tips_story`, `ref`

## Transactions

List individual earnings transactions, optionally filtered by type.

<CodeGroup>
```typescript TypeScript
// List transactions
const transactions = await client.access.earnings.listTransactions({
  startDate: '2024-01-01',
  type: 'tips',            // 'subscribes' | 'chat_messages' | 'post' | 'stream' | 'tips'
  tipsSource: 'chat',      // 'chat' | 'post_all' | 'profile' | 'story' | 'stream'
});

// Auto-paginate through all transactions
for await (const tx of client.access.earnings.iterateTransactions({
  type: 'tips',
})) {
  console.log(tx);
}
```

```python Python
# List transactions
transactions = earnings.list_transactions(
    client,
    start_date="2024-01-01",
    type="tips",
    tips_source="chat",
)

# Auto-paginate through all transactions
for tx in earnings.iter_transactions(client, type="tips"):
    print(tx)
```

```go Go
transactions, err := client.ListAccessEarningsTransactions(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "startDate":  []string{"2024-01-01"},
        "type":       []string{"tips"},
        "tipsSource": []string{"chat"},
    },
})
```

```csharp C#
var transactions = await client.GetAsync<EarningsTransactionsResponse>(
    "/v2/access/earnings/transactions",
    query: new Dictionary<string, string?>
    {
        ["startDate"] = "2024-01-01",
        ["type"] = "tips",
        ["tipsSource"] = "chat"
    }
);
```
</CodeGroup>

## Analytics

Content performance analytics are available across multiple categories. Each category offers a **chart** endpoint (time-series data) and a **top** endpoint (ranked content).

<Info>
  All analytics endpoints require the `analytics:read` permission.
</Info>

### Posts Analytics

<CodeGroup>
```typescript TypeScript
// Posts chart â€” time-series performance data
const postsChart = await client.access.analytics.posts.getChart({
  startDate: '2024-01-01',
  endDate: '2024-01-31',

  withTotal: true,

});

// Stats for a specific post
```
