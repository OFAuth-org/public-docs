## Official SDK

OFAuth provides a first-party Go SDK generated from our OpenAPI spec, so types and methods always match the latest API.

```bash
go get github.com/ofauth-org/onlyfans-sdk-go
```

### Hello, OFAuth

Verify your API key with a single call:

```go
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/ofauth-org/onlyfans-sdk-go"
)

func main() {
    client := ofauth.NewClient("your-api-key")

    acc, err := client.Whoami(context.Background())
    if err != nil {
        log.Fatal(err)
    }

    fmt.Println(acc.Id, acc.Permissions)
}
```

## Feature Highlights

<CardGroup cols={3}>
  <Card title="Full Type Safety" icon="code">
    Every SDK is generated from the OpenAPI spec — response types, request params, and error codes are all typed.
  </Card>

  <Card title="Automatic Pagination" icon="arrows-rotate">
    Manual pagination with typed query parameters and `HasMore` flag for efficient offset-based iteration.
  </Card>

  <Card title="Proxy Support" icon="globe">
    Call any OnlyFans API endpoint directly through the OFAuth proxy when the typed methods don't cover your use case.
  </Card>

  <Card title="Media Upload" icon="cloud-arrow-up">
    Upload images and videos with automatic chunking and progress callbacks built into the SDK.
  </Card>

  <Card title="Webhook Verification" icon="shield-check">
    Verify incoming webhook signatures to ensure events are authentic — included out of the box.
  </Card>

  <Card title="Error Handling" icon="triangle-exclamation">
    The SDK returns typed `*APIError` values with status, code, and details for precise error handling.
  </Card>
</CardGroup>
