## Subscribers

### List Subscribers

Get a paginated list of subscribers with optional filtering by status, date range, and search query.

```go
subs, err := client.ListAccessSubscribers(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "type":  []string{"active"},
        "limit": []string{"50"},
    },
})
if err != nil {
    log.Fatal(err)
}
for _, sub := range subs.List {
    fmt.Printf("%v\n", sub)
}
```

**Filter options for `type`:** `all` (default), `active`, `expired`, `latest`

When using `type=latest`, you can also pass `startDate`, `endDate`, and `latestType` (`total`, `new`, `renewals`):

```go
latestSubs, err := client.ListAccessSubscribers(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "type":       []string{"latest"},
        "startDate":  []string{"2024-01-01"},
        "endDate":    []string{"2024-01-31"},
        "latestType": []string{"new"},
        "limit":      []string{"20"},
    },
})
```

### Paginating Subscribers

The Go SDK uses manual offset-based pagination:

```go
// Manual pagination with offset
offset := 0
limit := 20

for {
    subs, err := client.ListAccessSubscribers(ctx, &ofauth.RequestOptions{
        Query: url.Values{
            "type":   []string{"active"},
            "limit":  []string{fmt.Sprintf("%d", limit)},
            "offset": []string{fmt.Sprintf("%d", offset)},
        },
    })
    if err != nil {
        log.Fatal(err)
    }

    for _, sub := range subs.List {
        fmt.Printf("%v\n", sub)
    }

    if !subs.HasMore {
        break
    }
    offset += len(subs.List)
}
```

### Subscriber Actions

Update notes, apply discounts, and set custom names for individual subscribers.

```go
// Update subscriber note
_, err := client.SetAccessSubscribersNote(ctx, "12345", struct {
    Text string `json:"text"`
}{Text: "VIP subscriber"})

// Apply discount to subscriber
_, err = client.SetAccessSubscribersDiscount(ctx, "12345", struct {
    Discount int `json:"discount"`
    Months   int `json:"months"`
}{Discount: 50, Months: 3})
```

## Subscriptions

Subscriptions represent people *you* are subscribed to or following (the inverse of subscribers).

### List Subscriptions

```go
subs, err := client.ListAccessSubscriptions(ctx, &ofauth.RequestOptions{
    Query: url.Values{
        "type":  []string{"active"},
        "limit": []string{"20"},
    },
})
for _, sub := range subs.List {
    fmt.Printf("%v\n", sub)
}
```

### Subscription Counts

Get aggregated counts of subscriptions and subscribers by status.

```go
counts, err := client.GetAccessSubscriptionsCount(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Subscriptions: %+v\n", counts.Subscriptions)
fmt.Printf("Subscribers: %+v\n", counts.Subscribers)
```

### Get Subscription by ID

```go
subscription, err := client.GetAccessSubscriptions(ctx, "sub_123")
```

### Subscription History

View the renewal and change history for a specific subscription.

```go
history, err := client.GetAccessSubscriptionsHistory(ctx, "sub_123", &ofauth.RequestOptions{
    Query: url.Values{
        "all": []string{"1"},
    },
})
```
