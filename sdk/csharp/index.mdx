## Official SDK

OFAuth provides a first-party SDK generated from our OpenAPI spec, so types and methods always match the latest API.

```bash
dotnet add package OnlyFans.SDK
```

### Hello, OFAuth

Verify your API key with a single call:

```csharp
using OFAuth;

var client = new OFAuthClient("your-api-key");

var account = await client.GetAsync<WhoamiResponse>("/v2/account/whoami");
Console.WriteLine($"{account.Id} {string.Join(", ", account.Permissions)}");
```

## Feature Highlights

<CardGroup cols={3}>
  <Card title="Full Type Safety" icon="code">
    Every SDK is generated from the OpenAPI spec — response types, request params, and error codes are all typed.
  </Card>

  <Card title="Automatic Pagination" icon="arrows-rotate">
    Manual pagination with query parameters for full control over paging through results.
  </Card>

  <Card title="Proxy Support" icon="globe">
    Call any OnlyFans API endpoint directly through the OFAuth proxy when the typed methods don't cover your use case.
  </Card>

  <Card title="Media Upload" icon="cloud-arrow-up">
    Upload images and videos with automatic chunking and progress callbacks built into the SDK.
  </Card>

  <Card title="Webhook Verification" icon="shield-check">
    Verify incoming webhook signatures to ensure events are authentic — included out of the box.
  </Card>

  <Card title="Error Handling" icon="triangle-exclamation">
    The SDK throws typed exceptions (`OFAuthException`) with status, code, and details.
  </Card>
</CardGroup>

## C# SDK

| | C# |
| --- | --- |
| **Package** | `OnlyFans.SDK` |
| **Repository** | [onlyfans-sdk-csharp](https://github.com/OFAuth-org/onlyfans-sdk-csharp) |
| **Init** | `new OFAuthClient()` |
| **Architecture** | `GetAsync<T>(path)` with 473 model classes |
| **Types** | Generated classes with `JsonPropertyName` |
| **Pagination** | Manual (query params) |
| **Dependencies** | `System.Net.Http`, `System.Text.Json` |
| **Key Feature** | DI support, `CancellationToken` |
