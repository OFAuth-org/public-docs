## Subscribers

### List Subscribers

Get a paginated list of subscribers with optional filtering by status, date range, and search query.

```csharp
var subs = await client.GetAsync<SubscribersListResponse>(
    "/v2/access/subscribers",
    query: new Dictionary<string, string?>
    {
        ["type"] = "active",
        ["limit"] = "50"
    }
);
foreach (var sub in subs.List)
{
    Console.WriteLine(sub);
}
```

**Filter options for `type`:** `all` (default), `active`, `expired`, `latest`

When using `type=latest`, you can also pass `startDate`, `endDate`, and `latestType` (`total`, `new`, `renewals`):

```csharp
var latestSubs = await client.GetAsync<SubscribersListResponse>(
    "/v2/access/subscribers",
    query: new Dictionary<string, string?>
    {
        ["type"] = "latest",
        ["startDate"] = "2024-01-01",
        ["endDate"] = "2024-01-31",
        ["latestType"] = "new",
        ["limit"] = "20"
    }
);
```

### Paginating Subscribers

The C# SDK uses manual offset-based pagination:

```csharp
// Manual pagination with offset
int offset = 0;
int limit = 20;

while (true)
{
    var subs = await client.GetAsync<SubscribersListResponse>(
        "/v2/access/subscribers",
        query: new Dictionary<string, string?>
        {
            ["type"] = "active",
            ["limit"] = limit.ToString(),
            ["offset"] = offset.ToString()
        }
    );

    foreach (var sub in subs.List)
    {
        Console.WriteLine(sub);
    }

    if (!subs.HasMore) break;
    offset += subs.List.Count;
}
```

### Subscriber Actions

Update notes, apply discounts, and set custom names for individual subscribers.

```csharp
// Update subscriber note
await client.PutAsync<object>(
    "/v2/access/subscribers/12345/note",
    body: new { text = "VIP subscriber" }
);

// Apply discount to subscriber
await client.PutAsync<object>(
    "/v2/access/subscribers/12345/discount",
    body: new { discount = 50, months = 3 }
);

// Set custom display name
await client.PutAsync<object>(
    "/v2/access/subscribers/12345/custom-name",
    body: new { name = "My Custom Name" }
);
```

## Subscriptions

Subscriptions represent people *you* are subscribed to or following (the inverse of subscribers).

### List Subscriptions

```csharp
var subs = await client.GetAsync<SubscriptionsListResponse>(
    "/v2/access/subscriptions",
    query: new Dictionary<string, string?>
    {
        ["type"] = "active",
        ["limit"] = "20"
    }
);
foreach (var sub in subs.List)
{
    Console.WriteLine(sub);
}
```

### Subscription Counts

Get aggregated counts of subscriptions and subscribers by status.

```csharp
var counts = await client.GetAsync<SubscriptionsCountResponse>(
    "/v2/access/subscriptions/count"
);
Console.WriteLine($"Subscriptions: {counts.Subscriptions}");
Console.WriteLine($"Subscribers: {counts.Subscribers}");
```

### Get Subscription by ID

```csharp
var subscription = await client.GetAsync<SubscriptionResponse>(
    "/v2/access/subscriptions/sub_123"
);
```

### Subscription History

View the renewal and change history for a specific subscription.

```csharp
var history = await client.GetAsync<SubscriptionHistoryResponse>(
    "/v2/access/subscriptions/sub_123/history",
    query: new Dictionary<string, string?> { ["all"] = "1" }
);
```

## Pagination Patterns

The C# SDK uses manual offset-based pagination. Check `response.HasMore` and increment `offset` to page through results.

<Tip>
  Use the `HasMore` flag and `offset` query parameter to manually page through results.
</Tip>
