## Earnings Chart

Get time-series earnings data for a connection. Use `startDate` and `endDate` to specify a date range, and `by` to break down by earning type.

```csharp
var chart = await client.GetAsync<EarningsChartResponse>(
    "/v2/access/earnings/chart",
    query: new Dictionary<string, string?>
    {
        ["startDate"] = "2024-01-01",
        ["endDate"] = "2024-01-31",
        ["by"] = "total",
        ["withTotal"] = "true"
    }
);
```

**`by` parameter options:** `total`, `messages`, `tips`, `stream`, `post`, `subscribes`, `tips_profile`, `tips_post`, `tips_chat`, `tips_stream`, `tips_story`, `ref`

## Transactions

List individual earnings transactions, optionally filtered by type.

```csharp
var transactions = await client.GetAsync<EarningsTransactionsResponse>(
    "/v2/access/earnings/transactions",
    query: new Dictionary<string, string?>
    {
        ["startDate"] = "2024-01-01",
        ["type"] = "tips",
        ["tipsSource"] = "chat"
    }
);
```

## Chargebacks

List chargeback records with optional date range filtering.

```csharp
var chargebacks = await client.GetAsync<EarningsChargebacksResponse>(
    "/v2/access/earnings/chargebacks",
    query: new Dictionary<string, string?>
    {
        ["startDate"] = "2024-01-01",
        ["endDate"] = "2024-01-31",
        ["limit"] = "20"
    }
);
```

## Analytics

Content performance analytics are available across multiple categories. Each category offers a **chart** endpoint (time-series data) and a **top** endpoint (ranked content).

<Info>
  All analytics endpoints require the `analytics:read` permission.
</Info>

### Posts Analytics

```csharp
// Posts chart
var postsChart = await client.GetAsync<AnalyticsPostsChartResponse>(
    "/v2/access/analytics/posts/chart",
    query: new Dictionary<string, string?>
    {
        ["startDate"] = "2024-01-01",
        ["endDate"] = "2024-01-31",
        ["by"] = "views",
        ["withTotal"] = "true"
    }
);

// Top performing posts
var topPosts = await client.GetAsync<AnalyticsPostsTopResponse>(
    "/v2/access/analytics/posts/top",
    query: new Dictionary<string, string?>
    {
        ["by"] = "purchases",
        ["startDate"] = "2024-01-01",
        ["endDate"] = "2024-01-31",
        ["limit"] = "10"
    }
);
```

### Analytics Namespace Reference

| Category | C# Path |
|---|---|
| Posts chart | `/v2/access/analytics/posts/chart` |
| Posts top | `/v2/access/analytics/posts/top` |
| Post stats | `/v2/access/analytics/posts/{id}` |
| Stories chart | `/v2/access/analytics/stories/chart` |
| Streams chart | `/v2/access/analytics/streams/chart` |
| Mass messages | `/v2/access/analytics/mass-messages/chart` |
| Promotions | `/v2/access/analytics/promotions/chart` |

<Warning>
  When querying large date ranges, use reasonable `limit` values and pagination to avoid timeouts. The `startDate` and `endDate` parameters accept `YYYY-MM-DD` format strings.
</Warning>
