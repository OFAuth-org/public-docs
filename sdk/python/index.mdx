## Official SDK

OFAuth provides a first-party Python SDK generated from our OpenAPI spec, so types and methods always match the latest API.

```bash
pip install onlyfans-sdk
```

### Hello, OFAuth

Verify your API key with a single call:

```python
from onlyfans_sdk import OFAuthClient, account

client = OFAuthClient(api_key="your-api-key")

result = account.whoami(client)
print(result["id"], result["permissions"])
```

## Feature Highlights

<CardGroup cols={3}>
  <Card title="Full Type Safety" icon="code">

    Generators handle pagination automatically so you can `for ... in` over any list.
  </Card>

  <Card title="Proxy Support" icon="globe">
    Call any OnlyFans API endpoint directly through the OFAuth proxy when the typed methods don't cover your use case.
  </Card>

  <Card title="Media Upload" icon="cloud-arrow-up">
    Upload images and videos with automatic chunking and progress callbacks built into the SDK.
  </Card>

  <Card title="Webhook Verification" icon="shield-check">
    Verify incoming webhook signatures to ensure events are authentic — included out of the box.
  </Card>

  <Card title="Error Handling" icon="triangle-exclamation">
    The SDK throws typed exceptions (`OFAuthError`) with status, code, and details.
  </Card>
</CardGroup>

## SDK Details

| | Python |
| --- | --- |
| **Package** | `onlyfans-sdk` |
| **Repository** | [onlyfans-sdk-python](https://github.com/OFAuth-org/onlyfans-sdk-python) |
| **Init** | `OFAuthClient()` |
| **Architecture** | Functional modules + client |
| **Types** | Pydantic v2 models |
| **Pagination** | Generators (`for ... in`) |
| **Dependencies** | `httpx` |
| **Key Feature** | Functional API (`posts.list_posts(client)`) |
