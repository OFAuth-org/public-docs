<Info>
  **Prerequisites:** You need an [OFAuth account](https://app.ofauth.com) and an API key. Grab your key from [app.ofauth.com/platform/developers/keys](https://app.ofauth.com/platform/developers/keys).
</Info>

<Steps>

<Step title="Install the SDK">

Pick your language and install the package:

<CodeGroup>
```bash npm
npm install @ofauth/onlyfans-sdk
```

```bash pnpm
pnpm add @ofauth/onlyfans-sdk
```

```bash yarn
yarn add @ofauth/onlyfans-sdk
```

```bash pip
pip install onlyfans-sdk
```

```bash Go
go get github.com/ofauth-org/onlyfans-sdk-go
```

```bash .NET
dotnet add package OnlyFans.SDK
```
</CodeGroup>

</Step>

<Step title="Initialize the client">

Create a client with your API key:

<CodeGroup>
```typescript TypeScript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: process.env.OFAUTH_API_KEY!,
});
```

```python Python
from onlyfans_sdk import OFAuthClient

client = OFAuthClient(api_key="your-api-key")
```

```go Go
package main

import "github.com/ofauth-org/onlyfans-sdk-go"

func main() {
    client := ofauth.NewClient("your-api-key")
}
```

```csharp C#
using OFAuth;

var client = new OFAuthClient("your-api-key");
```
</CodeGroup>

<Tip>
  Never hard-code your API key. Use environment variables (`OFAUTH_API_KEY`) or a secrets manager in production.
</Tip>

</Step>

<Step title="Verify your API key">

Call `whoami` to confirm your key works:

<CodeGroup>
```typescript TypeScript
import { createOFAuthClient, OFAuthAPIError } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: process.env.OFAUTH_API_KEY!,
});

try {
  const account = await client.account.whoami();
  console.log('Account ID:', account.id);
  console.log('Permissions:', account.permissions);
} catch (error) {
  if (error instanceof OFAuthAPIError) {
    console.error('API Error:', error.status, error.message);
  }
  throw error;
}
```

```python Python
from onlyfans_sdk import OFAuthClient, OFAuthError, account

client = OFAuthClient(api_key="your-api-key")

try:
    result = account.whoami(client)
    print("Account ID:", result["id"])
    print("Permissions:", result["permissions"])
except OFAuthError as e:
    print(f"API Error {e.status}: {e}")
```

```go Go
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/ofauth-org/onlyfans-sdk-go"
)

func main() {
    client := ofauth.NewClient("your-api-key")

    acc, err := client.Whoami(context.Background())
    if err != nil {
        if apiErr, ok := err.(*ofauth.APIError); ok {
            fmt.Printf("API Error %d: %s\n", apiErr.Status, apiErr.Message)
        }
        log.Fatal(err)
    }

    fmt.Println("Account ID:", acc.Id)
    fmt.Println("Permissions:", acc.Permissions)
}
```

```csharp C#
using OFAuth;

var client = new OFAuthClient("your-api-key");

try
{
    var account = await client.GetAsync<WhoamiResponse>("/v2/account/whoami");
    Console.WriteLine($"Account ID: {account.Id}");
    Console.WriteLine($"Permissions: {string.Join(", ", account.Permissions)}");
}
catch (OFAuthException ex)
{
    Console.WriteLine($"API Error {ex.Status}: {ex.Message}");
}
```
</CodeGroup>

<Check>
  You should see your organization ID and permissions list. If you get a `401` error, double-check your API key.
</Check>

</Step>

<Step title="Fetch OnlyFans data">

Once you have a connection ID (from [Link](/quickstart#step-2-connect-an-onlyfans-account)), use the Access API to pull data from a connected OnlyFans account:

<CodeGroup>
```typescript TypeScript
import { createOFAuthClient } from '@ofauth/onlyfans-sdk';

const client = createOFAuthClient({
  apiKey: process.env.OFAUTH_API_KEY!,
  connectionId: 'conn_YOUR_CONNECTION_ID',
});

// Get the connected creator's profile
const profile = await client.access.self.get();
console.log('Username:', profile.name);

// Or pass connectionId per-request
const profile2 = await client.access.self.get({
  connectionId: 'conn_OTHER_CONNECTION',
});
```

```python Python
from onlyfans_sdk import OFAuthClient
from onlyfans_sdk import self as self_module  # 'self' is a Python keyword

client = OFAuthClient(
    api_key="your-api-key",
    connection_id="conn_YOUR_CONNECTION_ID",
)

# Get the connected creator's profile
profile = self_module.list_selfs(client)
print("Username:", profile["name"])
```

```go Go
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/ofauth-org/onlyfans-sdk-go"
)

func main() {
    client := ofauth.NewClient("your-api-key").
        WithConnectionID("conn_YOUR_CONNECTION_ID")

    // Get the connected creator's profile
    profile, err := client.GetAccessSelf(context.Background(), nil)
    if err != nil {
        log.Fatal(err)
    }

    fmt.Println("Username:", profile.Username)
}
```

```csharp C#
using OFAuth;

var client = new OFAuthClient("your-api-key")
    .WithConnectionId("conn_YOUR_CONNECTION_ID");

// Get the connected creator's profile
var profile = await client.GetAsync<SelfResponse>("/v2/access/self");
Console.WriteLine($"Username: {profile.Username}");
```
</CodeGroup>

<Check>
  You're now pulling live data from a connected OnlyFans account through OFAuth.
</Check>

</Step>

</Steps>
