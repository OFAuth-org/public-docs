## Overview

The Account API exposes high-level details about your organization and tools to manage your authenticated connections.

## Base URLs

- Account Metadata: `https://api.ofauth.com/v2/account`
- Connections: `https://api.ofauth.com/v2/connections`

## Core Endpoints

| Endpoint | Method | Purpose |
| -------- | ------ | ------- |
| `/account/whoami` | `GET` | Return platform name, ID, benefits, permissions, allowed origins, and rate limits |
| `/connections` | `GET` | List all authenticated connections |
| `/connections/{connectionId}` | `DELETE` | Delete/disconnect a connection |

## Authentication

All endpoints require your API key.

```http
apikey: YOUR_API_KEY
```
