The Access API provides managed endpoints for common operations and proxy endpoints for any OnlyFans API path. OFAuth handles request signing, session management, and retries automatically.

<Card title="Access Guide" icon="book" href="/reference/access">
  Complete guide with authentication, endpoints, media proxy, rate limiting, and troubleshooting
</Card>

## Quick Reference

| Type | Base URL | Use Case |
|------|----------|----------|
| Managed | `/v2/access/*` | Typed routes for users, posts, messages, vault |
| Proxy | `/v2/access/proxy/*` | Any OnlyFans API path |

## Authentication

```http
apikey: YOUR_API_KEY
x-connection-id: conn_abc123
```

**Required Scope:** `ACCESS`
