OnlyFans Events is a beta Realtime surface for receiving supported OnlyFans events in your webhook endpoint.

<Info>
  **Beta status:** The top-level webhook contract is stable, but the set of documented event categories may continue to expand over time.
</Info>

## What Is Stable

- `data.payloadKey` is the main key to use for filtering and routing
- `data.shapeSubtype` may be present for documented event categories
- `data.json` contains the event payload
- New event categories may be added during the beta

## How To Integrate

1. Verify the webhook signature on the raw request body.
2. Route incoming deliveries by `data.payloadKey`.
3. Parse `data.json` defensively.
4. Tolerate additional event categories appearing over time during the beta.
