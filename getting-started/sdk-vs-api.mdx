## Overview

OFAuth publishes language-native SDKs that wrap our Access and Link APIs with strong typing, retry strategies, and opinionated workflows. Use this directory to jump straight to the toolkit that fits your stack, whether you build with dynamic scripting languages or compiled backends.

<Info>
  Prefer to wire your own HTTP client? Head over to the [Access API guide](/guides/access) for request/response details and examples.
</Info>

## Shared Capabilities

- Consistent method naming and authentication helpers across languages.
- Built-in retry logic tuned for OnlyFans rate limiting policies.
- Structured error envelopes so you can triage without parsing raw bodies.
- Examples for Link handshakes, Access data pulls, and webhook validation.
