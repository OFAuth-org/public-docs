## API Overview

<Info>
	**New to APIs?** Don't worry! Our guides assume no prior knowledge and include
	complete code examples you can copy and paste.
</Info>

### Three Main APIs

- **[Link API](/guides/link/overview)**: Handle user authentication and
  connection management
- **[Access API](/guides/access)**: Seamless access to OnlyFans data with
  automatic session handling
- **[Account API](/guides/account)**: Manage webhooks, monitor connections, and
  configure settings

### Authentication Methods

- **Hosted Mode**: Users authenticate on OFAuth's secure pages (recommended for
  beginners)
- **API Mode**: Full control over authentication UX (requires approval)
- **Session Headers**: Use your own OnlyFans sessions with Access API
